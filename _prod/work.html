Tiny Terms

Overview

Tiny Terms is an edducational app I worked on in my spare time, it provides a way to learn a subject using cards which contain useful information about a specific term. Users can upload their own lists, customise cards using colours and symbols as well as use built in integration with Wikipedia and Glosbe translations to take learning to a high level. This was a solo project and has found success on the Google Play store and as a web app.

Roles

  • Design and concept
  • Development
  • Marketing and release management

Length of project

8 months

Tech stack

  • Pure ES6, Pug, SCSS
  • Gulp, Babel, Rollup
  • Cordova

Libraries

  • Tabletop
  • localForage

Key project features

  • Solo project, from concept through to release
  • Lightweight, responsive SPA
  • Focus on UX and highly effective study techniques

Visit the site

https://play.google.com/store/apps/details?id=com.phonegap.tinytermshttps://github.com/alexplummer/language-app
